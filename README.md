## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/RTBRuhan/WealthProjector.git

# Navigate to project directory
cd WealthProjector

# Install dependencies
npm install

# Start development server

```

The app will be available at `http://localhost:3000`

